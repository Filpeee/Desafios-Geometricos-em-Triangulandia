# Desafios Geométricos em Triangulândia

## Sobre o Projeto
Este repositório contém as soluções para dois desafios de otimização focados em formas geométricas (triângulos) para planejamento urbano.

## Algoritmos e Estruturas de Dados
O trabalho explora dois paradigmas fundamentais de algoritmos:
* **Parte 1 (O Muro):** O objetivo é determinar a altura máxima de um muro em formato de triângulo isósceles esculpido a partir de pilhas de blocos. A solução utiliza Programação Dinâmica com uma abordagem gulosa para encontrar a maior estrutura possível.
* **Parte 2 (As Árvores):** Busca identificar três árvores no plano cartesiano que formem o triângulo de menor perímetro. A solução aplica o paradigma de Divisão e Conquista, sendo uma adaptação do problema clássico do "Par de Pontos Mais Próximos".

## Complexidade
* **Parte 1:** Tempo O(N) e Memória O(N), onde N é o número de pilhas.
* **Parte 2:** Tempo O(Z * log^2 Z) e Memória O(Z), onde Z é o número de árvores.
