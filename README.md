# Desafios Geométricos em Triangulândia

## Sobre o Projeto
[cite_start]Este repositório contém as soluções para dois desafios de otimização focados em formas geométricas (triângulos) para planejamento urbano[cite: 83, 84].

## Algoritmos e Estruturas de Dados
O trabalho explora dois paradigmas fundamentais de algoritmos:
* [cite_start]**Parte 1 (O Muro):** O objetivo é determinar a altura máxima de um muro em formato de triângulo isósceles esculpido a partir de pilhas de blocos[cite: 85, 86]. [cite_start]A solução utiliza Programação Dinâmica com uma abordagem gulosa para calcular as maiores rampas ascendentes e descendentes em um histograma[cite: 93, 107, 108, 119].
* [cite_start]**Parte 2 (As Árvores):** Busca identificar três árvores no plano cartesiano que formem o triângulo de menor perímetro[cite: 88, 89]. [cite_start]A solução aplica o paradigma de Divisão e Conquista, sendo uma adaptação do problema clássico do "Par de Pontos Mais Próximos"[cite: 102].

## Complexidade
* [cite_start]**Parte 1:** Tempo $O(N)$ e Memória $O(N)$, onde $N$ é o número de pilhas[cite: 134, 137, 138].
* [cite_start]**Parte 2:** Tempo $O(Z~log^{2}Z)$ e Memória $O(Z)$, onde $Z$ é o número de árvores[cite: 134, 143, 145].
